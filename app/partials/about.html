Understanding the flow

This is very simple flow. In which any transaction start with amount. Following are 5 attribute of the transaction

  • Amount: all transaction must have amount
  • Type: only 3 type..Income..Expense...Transfer
  • Source: depending on type of transaction, source could be different. e.g Source of expense might be shopping and destination credit card
  • Destination: Same as source ex..one of source of income might be Salary and destination bank saving account
  • Date: This is inherent character of any activity
source and destination are just endpoint of any money activity flow. will give you use case scenario in follwoing table. so you can understand identify the transaction type, source and destination to use this app efficently.
